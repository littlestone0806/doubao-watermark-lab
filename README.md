# 水印清理工作台

一个使用 Electron 封装豆包网页版的批量图片去水印工具。登录会话存放在独立的持久化 Electron 分区中，应用重启后会继续使用上次登录状态。

## 功能

- 在独立豆包窗口中登录，自动保留 Cookie、LocalStorage 等会话数据。
- 拖拽或批量选择图片，逐张上传并发送可编辑提示词。
- 同时观察豆包页面图片节点与网络图片请求，优先下载分辨率最高的原始资源候选。
- 对可信图片 CDN 的图片处理 / 水印参数尝试直接原始链接。
- 在未找到可信原图时，可让豆包预留底栏并自动裁掉指定百分比。
- 每张图片可使用新对话，支持任务间隔、错误隔离、停止任务与输出定位。
- 可切换是否显示豆包处理窗口；调试阶段默认显示，稳定后可关闭并在后台运行。

## 本地运行

```bash
npm install
npm start
```

首次使用：

1. 点击“登录 / 打开豆包”，在打开的豆包网页中完成登录。
2. 返回主窗口添加图片并选择输出目录。
3. 确认处理策略后开始批量处理。任务期间保持豆包窗口开启。

## 打包

```bash
npm run dist
```

构建产物位于 `dist/`。macOS、Windows 和 Linux 的具体产物取决于执行打包命令的平台。

## 实现说明与限制

豆包没有为这一网页流程提供稳定的公开 DOM 自动化接口，页面更新后上传按钮、输入框或发送按钮可能发生变化。本项目使用语义、位置和多选择器回退寻找控件，核心逻辑位于 `src/doubao-automation.js`，便于后续维护。

“原始资源候选”表示应用成功获得了不含已知图片处理或水印参数的大图链接，并不构成对图片内容中绝对无水印的保证。应用默认在顶部增加 10% 的临时隔离带，让豆包 AI 标识落在隔离带内；生成完成后按原图宽高比精确裁除，并额外使用 0.5% 的白边补偿。右侧齿轮可调整提示词、顶部/底部、隔离带比例和补偿比例。请只处理你拥有或获授权编辑的图片，并遵守豆包的服务规则。
